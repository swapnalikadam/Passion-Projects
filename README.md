# Resposive Website using HTML5 and CSS3

After graduating, I pursued my passion for web development by designing a website to align with industry standards. Following research, I embraced HTML5 for markup and CSS3 for styling, recognizing their trendiness in web development. Ensuring responsiveness for both desktop and phone screens became a primary focus.

In my exploration, I utilized key HTML elements like '<header>', '<footer>', and '<section>' for effective content organization. Additionally, I incorporated CSS3 features such as border-radius to enhance design aesthetics. To optimize the website for various screen sizes, I implemented a responsive background image, ensuring a seamless experience across different interfaces.
